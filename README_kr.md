# 비쥬얼 스튜디오 코드 - 오픈 소스 ('Code - OSS')

## 저장소

이 저장소('Code - OSS')는 소프트웨어의 오픈 소스 버전으로서, Microsoft가 커뮤니티와 함께 개발하는 곳입니다. 코드와 이슈 작업 외에도 로드맵, 월간 계획, 종료 계획 등을 게시합니다. 이 소스 코드는 표준 MIT 라이센스에 따라 모두에게 개방되어 있습니다.

## 비쥬얼 스튜디오 코드

비주얼 스튜디오 코드는 간결한 코드 편집기의 장점과 개발자가 필요로 하는 코드 편집, 빌드, 디버그 사이클을 결합한 기능을 제공합니다. 매달 새로운 기능과 버그 수정이 업데이트됩니다. Windows, macOS, Linux용으로 다운로드할 수 있으며, 최신 기능을 원하면 인사이더 빌드를 설치할 수 있습니다.

## 기여

다양한 방법으로 프로젝트에 참여할 수 있습니다. 버그 및 기능 요청 제출, 소스 코드 변경 검토, 문서 수정 등의 방법이 있습니다. 직접 코드에 기여하려면 기여 방법 문서를 참조하세요.

## 피드백

Stack Overflow에서 질문하거나, 새로운 기능을 요청하거나, 인기 있는 기능 요청에 투표하거나, 이슈를 보고할 수 있습니다. Twitter에서 @code를 팔로우하여 피드백을 보내주세요.

## 관련 프로젝트

VS Code의 주요 구성 요소와 확장 기능들은 GitHub의 별도 저장소에 있으며, 관련 프로젝트 페이지에서 확인할 수 있습니다.

## 포함된 확장 기능

VS Code는 여러 언어에 대한 구문 강조 및 스니펫을 포함한 확장 기능을 내장하고 있습니다.
